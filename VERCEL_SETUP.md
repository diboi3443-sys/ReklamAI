# Настройка Vercel для исправления черного экрана

## Проблема
Черный экран на Vercel обычно возникает из-за:
1. Отсутствия переменных окружения
2. Ошибок JavaScript, которые блокируют рендеринг
3. Неправильной конфигурации build

## Решение

### 1. Установите переменные окружения в Vercel

1. Откройте проект в [Vercel Dashboard](https://vercel.com/dashboard)
2. Перейдите в **Settings** → **Environment Variables**
3. Добавьте следующие переменные:

```
VITE_SUPABASE_URL=https://your-project.supabase.co
VITE_SUPABASE_ANON_KEY=your-anon-key-here
```


4. Выберите **Production**, **Preview**, и **Development**
5. Нажмите **Save**

### 2. Пересоберите проект

После добавления переменных:
1. Перейдите в **Deployments**
2. Найдите последний deployment
3. Нажмите **Redeploy** (три точки → Redeploy)
4. Или сделайте новый commit в GitHub - Vercel автоматически пересоберет

### 3. Проверьте консоль браузера

Откройте сайт и нажмите F12:
- Если видите ошибки о переменных окружения - они не установлены
- Если видите другие ошибки - проверьте их в консоли

### 4. Проверьте build логи

В Vercel Dashboard → Deployments → выберите deployment → **Build Logs**:
- Убедитесь, что build прошел успешно
- Проверьте, что нет ошибок компиляции
